# Financial-Dashboard
## Project Objective 
Develop a clear and interactive dashboard designed to help the company better visualize and understand its investor profiles.
## Dataset used
- <a href="https://github.com/CamillySR/Financial-Dashboard/blob/main/Finance_data.xlsx">Financial Dashboard Data</a>
## Questions 
- What is the predominant investor profile?

- Which investments are the highest rated?

- Is there a relationship between financial goals and investment type?

- What is the most common investment horizon?

How do investors gather information before investing?
Financial dashboard project developed in Power BI, focused on analyzing investor profiles.
